O Visual Studio Code, mais conhecido como VSCode, é um editor de código-fonte gratuito e multiplataforma desenvolvido pela Microsoft. Ele é amplamente utilizado por desenvolvedores devido à sua interface amigável, extensibilidade e robustez.

Instalação e Configuração Inicial

Download e Instalação: Acesse Visual Studio Code e baixe a versão apropriada para o seu sistema operacional (Windows, macOS ou Linux). Siga as instruções do instalador para concluir a instalação.

Primeira Execução: Ao abrir o VSCode pela primeira vez, você verá a tela de boas-vindas, que oferece links rápidos para abrir pastas, arquivos, personalizar o layout e acessar documentações.

O Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991.

Instalação e Configuração Inicial

Download e Instalação: Baixe o instalador adequado para seu sistema operacional em python.org. Execute o instalador e siga as instruções na tela para completar a instalação.

Verificando a Instalação

Finalizando instalação: Verifique se o Python foi instalado corretamente, para fazer isso primeira abra o terminal ctrl+shit+t apos isso digite o seguinte comando no seu terminal:

                
        $ python3 --version

Devera aparecer o numero da versão do python que foi instalada, caso contrario volte aos passos anteriores e os refaça.

                
        $ python 3.12.2

bootstrap é um framework CSS open-source criado pela equipe do Twitter. Ele fornece uma coleção de ferramentas para criar interfaces de usuário com design consistente e responsivo. Isso significa que seus sites e aplicativos serão automaticamente ajustados para diferentes tamanhos de tela, como desktops, tablets e smartphones. .
